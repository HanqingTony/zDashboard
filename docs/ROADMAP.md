# zDashboard 路线图

## v0.2.0（计划中）

### 资源管理模块
- 接入 MinIO，在线管理文件（浏览、上传、下载、删除）
- 文件元信息组织数据库：标签、分类、描述、关联关系
- 替代目前通过文件路径硬编码的音频播放方式

### 英语学习模块增强
- 单独的单词本页面：按状态筛选、搜索、批量操作
- 单词本独立于文章，可手动添加/删除单词

### 文章分析模块
- 一键全部标白按钮：将当前文章所有单词标记为 mastered

### 备份模块
- 一键备份：导出 zdb.db + 音频文件 + MinIO 数据
- 备份监控：显示最近备份时间、备份大小、自动提醒

### LLM 语音模块改进
- 将 TTS 模型编译为 WASM，直接在前端运行
- 去除对 ComfyUI 后端的依赖，实现离线语音合成

### 项目管理页面
- 监视仓库/分支/提交编号
- 展示各项目部署状态和版本信息
